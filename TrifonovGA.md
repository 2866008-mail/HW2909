# Базовые команды GIT #

## init ## 

**git init** - создает новый репозиторий Git. 

Пример выполняется в папке репозитория: 
```
git init
```

## add ## 

**git add** -  добавляет указанные файлы в индекс

Пример выполняется в папке репозитория: 
```
git add newFile.txt
```

## commit ## 

**git commit** - команда для записи индексированных изменений в репозиторий

Пример выполняется в папке репозитория: 
```
git commit -m 'comment text'
```

## checkout ## 

**git checkout** - позволяет перемещаться между ветками, коммитами

Пример выполняется в папке репозитория: 

*перемещение на заданный коммит при указании части хеша*
```
git checkout 4ba4
```
Где "4ba4" - четыре первых символа хеша коммита

*создание ветки и перемещение на созданную ветку new_branch*
```
git checkout -b new_branch
```

## log ## 

**git log** - историю проекта

Пример выполняется в папке репозитория: 

*вывод истории*
```
git log
```

*вывод истории с графическим дополнением*
```
git log --graph
```

## branch ## 

**git branch** - работа с ветками

Пример выполняется в папке репозитория: 

*вывести список веток*
```
git branch
```

*удаление ветки name_branch*
```
git branch -d name_branch
```

## merge ## 

**git merge** - слияние веток

Пример выполняется в папке репозитория: 

*слить указанную ветку (work_branch) с текущей*
```
git merge work_branch
```

## clone ##

**git clone** - создание копии (удаленного) репозитория

Пример выполняется в папке репозитория: 

*клонируется удаленный репозиторий с url git@github.com:Griffon2012/HW2909.git*
```
git clone git@github.com:Griffon2012/HW2909.git
```

## push ##

**git push** - используется для выгрузки содержимого локального репозитория в удаленный репозиторий

Пример выполняется в папке репозитория: 

```
git push
```

## pull ##

**git pull** - используется для извлечения и загрузки содержимого из удаленного репозитория и немедленного обновления локального репозитория этим содержимым

Пример выполняется в папке репозитория: 

```
git pull
```
Полное описание команд [Git](https://git-scm.com/docs/user-manual "git documentation")
